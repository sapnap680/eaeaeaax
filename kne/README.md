# カード発行システム

仮選手証・スタッフ証を発行するWebアプリケーション

## 機能

- 氏名、大学名、役職（選手/スタッフ）の入力
- 顔写真のアップロード
- 大会情報の設定（回数、種別：新人戦/選手権大会/リーグ戦）
- 有効期限の設定（今大会のみ or 特定日付）
- カード画像の生成とダウンロード

## セットアップ

```bash
npm install
```

## 開発

```bash
npm run dev
```

## デプロイ

Vercelにデプロイ可能

1. GitHubにプッシュ
2. Vercelでリポジトリをインポート
3. 自動デプロイ
